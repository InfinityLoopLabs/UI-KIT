# UI-KIT

UI kit scaffolded with Vite, Tailwind CSS v4, and shadcn/ui support.

## Requirements

- Node.js 18+

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Library output

The build generates a library bundle in `dist/` with the following formats:

- ES module (`dist/ui-kit.js`)
- CommonJS (`dist/ui-kit.cjs`)
- UMD (`dist/ui-kit.umd.cjs`)
